# Job Application Tracker - Web App Pribadi

This is a personal web application to track job applications.

## Tech Stack
- **Frontend**: Vite + React + TypeScript, Tailwind CSS, Recharts
- **Backend**: Node.js + Express.js + TypeScript
- **Database**: MySQL

## Prerequisites
- Node.js v18+
- XAMPP (for MySQL)

## Installation Guide

1. Activate MySQL in XAMPP
2. Import `database.sql` into phpMyAdmin (`job_tracker` database)
3. Open a terminal and run the backend:
   ```bash
   cd backend
   npm install
   npm run dev
   ```
4. Open another terminal and run the frontend:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
5. Open http://localhost:5173 in your browser
